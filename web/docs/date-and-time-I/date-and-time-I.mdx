# Date and Time I

## Find the Day of the Week

Write a function that takes a date and returns the date with the day of the week in the following format: `Monday, January 20, 2024`.

How to solve this problem:

-   Create an array containing the days of the week starting with `Sunday`.

1. Use the `Date` object to get the day of the week.
2. Use the `Date` object to get the month.
3. Use the `Date` object to get the day of the month.
4. Use the `Date` object to get the year.
5. Return the formatted date.
