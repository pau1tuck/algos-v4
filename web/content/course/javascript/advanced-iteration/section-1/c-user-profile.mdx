-   EXPLAIN: Real-world scenario: This simulates processing data from a user management system.


### Instructions

Given a JavaScript object containing basic profile data (e.g., name, email, etc.), iterate over its properties and display a formatted summary.

#### Input

A `user` <Blue>**object**</Blue> containing user profile data.

```JavaScript
const user = {
    name: "John Doe",
    email: "jdoe@me.com",
    phone: "555-555-5555",
}
```


#### Output

A <Blue>**string**</Blue> containing a formatted, multiline summary of the user's profile data.
```JavaScript
John Doe
jdoe@me.com
555-555-5555
```

#### Notes

- You should use a `for...in` loop to iterate through the object's properties.
- You will need to use template literals to format the output.
